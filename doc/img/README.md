Image processing library in C++ {#mainpage}
===========================================

Dmitigr Img (hereinafter referred to as Img) - is a simple library
to work with raster images in C++. Img is a part of the
[Dmitigr Cefeika][dmitigr_cefeika] project.

**ATTENTION, this software is "alpha" quality, and the API is a subject to change!**

Documentation
=============

The [Doxygen]-generated documentation is located [here][dmitigr_img_doc].

Features
========

Currently it's just a tiny wrapper around the [GraphicsMagick] library.
